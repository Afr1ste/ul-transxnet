# Model Zoo

Trained PyTorch checkpoints (`*.pth`, `*.pt`), YOLO weights, ONNX exports, and Android APKs are intentionally excluded from this repository.

Recommended release policy:

- Put final model weights in GitHub Releases, Hugging Face, Zenodo, or an institutional archive.
- Keep intermediate checkpoints and failed experimental artifacts out of version control.
- Record SHA256 checksums and expected metrics for any released model binary.

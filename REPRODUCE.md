# Reproduction Notes

This package is organized for table-level reproducibility and provenance audit.

## Environment

The manuscript experiments were run in local Python environments with PyTorch, timm, scikit-learn, pandas, NumPy, Pillow, matplotlib, and ultralytics for detector experiments. `requirements.txt` gives a compact package list for reading and regenerating summary artifacts.

## Regenerate No-Retrain Revision Tables

From the original local project layout, the current no-retrain tables are generated by:

```powershell
python src/scripts/generate_no_retrain_revision_tables.py
```

The public repository stores the resulting CSV/TeX artifacts in:

```text
results/no_retrain_revision_20260505/
```

## Regenerate High-ROI No-Retrain Diagnostics

The additional no-retrain diagnostic tables and reliability diagram are generated from frozen prediction/result CSV files only:

```powershell
python src/scripts/generate_high_roi_no_retrain_tables.py
```

The public repository stores the generated artifacts in:

```text
results/high_roi_no_retrain_20260505/
paper/figures/fig_reliability_diagram_case_level.png
```

The source CSVs used by the model-selection audit are copied into:

```text
results/frozen_source_logs/
```

## Model Selection Audit

The final model-family row is selected by highest mean validation AUC across TN5000, BUSI, and AUL under dataset-specific training. See:

```text
MODEL_SELECTION_PROTOCOL.md
results/no_retrain_revision_20260505/validation_selection_audit.csv
results/no_retrain_revision_20260505/model_selection_source_files.csv
```

## Manuscript Build

The manuscript was compiled with `latexmk`:

```powershell
cd paper
latexmk -pdf -interaction=nonstopmode main.tex
```

The checked-in PDF is `paper/main.pdf`.

## Full Training Boundary

Full retraining is not directly runnable from this repository alone because dataset image files, generated ROI folders, checkpoints, detector weights, ONNX binaries, APKs, and raw logs are not redistributed. Scripts are included to document the exact training/evaluation entry points, but dataset reconstruction must follow the original dataset licenses and download instructions.

# Reproduction Notes

This repository supports manuscript-level audit rather than full dataset redistribution.

## Validate the Public Artifact

```powershell
python scripts/validate_artifact.py
```

The validator checks that required files exist, that no obvious stale internal experiment names remain in public manuscript text, and that public CSV files do not contain absolute local Windows paths.

## Export Manuscript Tables

```powershell
python scripts/reproduce_main_tables.py --out-dir reproduced_tables
```

This copies the table-level CSVs from `results/strict_20260514/manuscript_tables/` into a separate folder for inspection.

## Rebuild the Manuscript

Install a recent TeX distribution with `latexmk`, then run:

```powershell
cd paper
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
latexmk -pdf -interaction=nonstopmode -halt-on-error supplementary_material.tex
```

The committed PDFs were built from the same source package.

## Regenerate Clean Figures

The submitted architecture, ROI robustness, and mobile trade-off figures were regenerated by:

```powershell
python tools/make_clean_submission_figures.py
```

The script reads the compact strict result CSVs committed under `results/strict_20260514/`. The committed figure PDFs are included for review.
